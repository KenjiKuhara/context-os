## 3. 手動 E2E 手順（DoD #1〜#5 チェックリスト）

| # | 確認項目 | 手順 | 結果 |
|---|----------|------|------|
| 1 | **種別フィルタで絞り込める** | 種別セレクトで「関連」を選び「絞り込む」を押す。→ relation のみ表示される。「すべて」に戻して「絞り込む」で全種別が表示される。 | ✅ |
| 2 | **node_id（UUID）で絞り込める** | 履歴に含まれる node_id を入力欄に入れ「絞り込む」を押す。→ その Node に関係する履歴のみ表示される。 | ✅ |
| 3 | **不正 UUID は API を呼ばずにエラー表示** | node_id に `invalid` 等を入力し「絞り込む」を押す。→ 「UUID形式ではありません」が表示され、一覧は変更されない（API 未呼び出し）。 | ✅ |
| 4 | **0 件時メッセージ「該当する履歴がありません」** | 存在しない node_id（例: 00000000-0000-0000-0000-000000000000）で「絞り込む」を押す。→ 「該当する履歴がありません」と表示される。 | ✅ |
| 5 | **Apply 後にリロード不要で履歴が増える（フィルタ条件維持）** | フィルタをかけた状態で relation/grouping/decomposition のいずれかを Apply する。→ 成功後、同じフィルタ条件のまま履歴が 1 件増えて表示される。F5 は不要。 | ✅ |
